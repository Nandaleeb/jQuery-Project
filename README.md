# Simple jQuery Website

A clean, responsive website built with HTML, CSS, jQuery, and Font Awesome icons.

## Features

- **5 Pages**: Home, About, Services, Contact, Weather
- **jQuery Effects**: Animations, form validation, modals
- **Real Weather API**: Live weather data
- **Responsive Design**: Works on all devices
- **Font Awesome Icons**: Professional icons
- **Separate Files**: Each page has its own CSS and JS

## Structure

```
/css/          - Stylesheets for each page
/js/           - JavaScript files for each page
home.html     - Home page
about.html     - About page
services.html  - Services page
contact.html   - Contact page
weather.html   - Weather page
```

## Usage

1. Open `home.html` in your browser
2. Navigate between pages using the menu
3. Try the weather search functionality
4. Test the contact form validation

## Technologies

- HTML5
- CSS3
- jQuery 3.6.0
- Font Awesome 6.0
- OpenWeatherMap API
